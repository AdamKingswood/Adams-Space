# Adam's Space

A personal learning space documenting my journey in web development.

## About

This is my first solo web development project - a place to practice HTML, CSS and JavaScript fundamentals while building something meaningful. The site serves as both a portfolio where I document my progress, reflections, and projects.

## Features

- **Responsive Design** - Works on all screen sizes
- **Accessibility** - Skip links, semantic HTML, ARIA labels
- **CSS Grid Layout** - Project showcase using modern CSS Grid
- **Custom CSS Variables** - Organized, maintainable styling system
- **Scroll Animations** - Smooth reveal effects using IntersectionObserver

## Technologies

- HTML5
- CSS3 (Grid, Custom Properties, Animations)
- JavaScript (IntersectionObserver API)

## Pages

- **Home** - Landing page with introduction
- **Who I Am** - About me
- **How My Work Shapes My Learning** - Reflections on learning while working retail
- **Projects** - Showcase of web development projects
- **Reflections** - Thoughtful entries about the learning process
- **Learning Log** - Daily progress tracking

## Learning Approach

This project is built using guided learning - understanding each concept before implementation rather than copy/pasting code. The focus is on sustainable learning and self-sufficiency.
